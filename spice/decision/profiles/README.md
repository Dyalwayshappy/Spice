# Decision Profiles

This package contains read-only starter templates for user-facing `decision.md` configuration.

- `default.decision.md`: conservative default decision guidance template
- `default_support.json`: reference support contract for explain/demo/debug flows

Copy templates into a project-local path before editing:

```sh
python -m spice.entry decision init
```

The bundled defaults are never mutated by Spice.
